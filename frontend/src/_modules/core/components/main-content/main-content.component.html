Sistema de Finanças

Código Livre

APRESENTAÇÃO

A premissa deste projeto é garantir que quem quer queira possa ter um sistema a executar onde sentir-se mais seguro.

A ideia do sistema é permitir que qualquer pessoa possa ter um sistema de educação financeira e controle de finanças pessoais, apesar de ser útil para pequenas empresas, pois é mono-usuário, ou seja, cada usuário possui suas próprias contas e lançamentos.
