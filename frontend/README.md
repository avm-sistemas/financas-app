
# FINANÇAS App


## DEFINIÇÃO

O frontend foi desenvolvido em Angular 17.


## CONFIGURAÇÃO

```bash
    > copy .env.sample .env    
```

Editar o arquivo .env e indicar as URLs dos serviços API-CONTROLE e API-CONSOLIDADO


```bash
    > npm install -g @angular/cli@17

    > cd frontend

    > npm install
```

## COMPILAÇÃO

```bash
    > npm run build
```

## EXECUÇÃO

```bash
    > ng serve
```
